About This HTTP Server

This is a simple HTTP server built using Java's ServerSocket. It listens for incoming connections and serves static HTML files.

When a client requests /About.html, the server reads this file and returns it as a response.

Example:

$ curl -v http://localhost:4221/About.html
HTTP/1.1 200 OK
Content-Type: text/html
(Content of this page)

To add more pages, simply place additional HTML files in the server's root directory.
